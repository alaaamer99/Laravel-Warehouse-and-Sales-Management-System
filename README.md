## 🚀 Installation

```bash id="inst_warehouse_01"
git clone https://github.com/alaaamer99/Laravel-Warehouse-and-Sales-Management-System.git
cd Laravel-Warehouse-and-Sales-Management-System

composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan storage:link
php artisan serve
